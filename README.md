# pyTeachConnect 网站

这是 pyTeachConnect 项目的官方网站，基于 GitHub 仓库 https://github.com/pyTeachConnect 构建。

## 项目介绍

pyTeachConnect 是一个基于 Python 开发的教师通信平台，致力于连接教室与办公室的教师，实现信息的快速传达与协作。

## 网站功能

- **首页**：项目简介和快速导航
- **关于我们**：项目背景和团队介绍
- **功能特性**：展示平台核心功能
- **项目展示**：展示相关项目和案例
- **联系我们**：联系方式和反馈渠道

## 技术栈

- **HTML5**：网站结构
- **CSS3**：样式设计
- **JavaScript**：交互功能
- **响应式设计**：适配不同设备

## 快速开始

### 本地运行

1. 进入 web 目录
2. 启动本地服务器：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   ```
3. 在浏览器中访问：`http://localhost:8000`

## 项目结构

```
web/
```

## 开发说明

本网站采用纯前端技术开发，无需后端服务器支持。可以直接部署到任何静态网站托管服务。

### 样式设计

- 使用渐变色和现代化设计风格
- 响应式布局，适配桌面和移动设备
- 简洁清晰的导航结构
- 丰富的交互动效

### JavaScript 功能

- 平滑滚动
- 导航栏滚动效果
- 功能卡片悬停动画
- 页面加载动画
- 返回顶部按钮
